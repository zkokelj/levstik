# Levstikova aleja nagrajencev

Website for Levstik lifetime achievement award winners.

## Technologies used

- Laravel
- Bulma CSS
- MySql
- JavaScript

## Website features
- Multi-language support
- Responsive design
- CRUD operations supported
- Registration & login system
